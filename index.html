שאלה 1: מהו ISP?









ISP (Internet Service Provider) הוא הגוף שמחבר את המשתמש לאינטרנט – כמו בזק, נטוויז'ן וכו'.

שאלה 2: מהו פינג (Ping)?









פינג מודד את זמן ההשהיה (latency) בין שליחת בקשה לשרת לבין קבלת תגובה.

שאלה 3: מהי כתובת IP?









כתובת IP (Internet Protocol) היא כתובת מספרית ייחודית שמוקצית לכל מכשיר ברשת על מנת לאפשר תקשורת בין מכשירים.

שאלה 4: מהי רוחב פס (Bandwidth)?









רוחב פס הוא מדד לקיבולת התעבורה – כמה נתונים אפשר להעביר בזמן מסוים.

שאלה 5: מהי כתובת URL?









URL (Uniform Resource Locator) הוא הנתיב המלא למידע באינטרנט, כולל פרוטוקול, דומיין ונתיב.

שאלה 6: מה תפקידו של הראוטר בבית?









הראוטר מקשר בין הרשת הביתית לרשת האינטרנט, מחלק כתובות IP פנימיות ומאפשר גלישה.

שאלה 7: מהו DNS?









DNS (Domain Name System) ממיר שמות דומיין כמו google.com לכתובות IP כדי לאפשר התחברות לשרתים.

שאלה 8: מהי כתובת MAC?









כתובת MAC (Media Access Control) היא מזהה ייחודי שניתן לכרטיס רשת של כל מכשיר.

שאלה 9: מה ההבדל בין HTTP ל-HTTPS?









HTTPS מוסיף שכבת הצפנה (SSL/TLS) על HTTP כדי להגן על המידע שמועבר.

שאלה 10: מהו פרוטוקול HTTP?









HTTP (Hypertext Transfer Protocol) הוא פרוטוקול המשמש להעברת תוכן באינטרנט – בעיקר דפי אינטרנט.
